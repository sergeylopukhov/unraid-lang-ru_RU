:ud_edit_script_device_settings_plug:
> Выберите параметры для этого устройства / точки монтирования. Эти изменения вносятся немедленно. Вам не нужно нажимать «Сохранить», чтобы сохранить их.
:end

:ud_disk_name_plug:
> Введите псевдоним для этого устройства. Это имя, которое вы хотите использовать для ссылки на этот диск. Оставьте имя пустым, и будет назначен «devX».
>
> **Название:** не более 8 символов, пробелы преобразованы в '_', минимальное количество специальных символов разрешено, имя чувствительно к регистру.
>
> **Примечание.** Не может содержать " dev " как первые символы. Это зарезервировано для устройств «devX».
:end

:ud_apfs_volume_plug:
> Введите том для установки в этом разделе apfs. Нажмите **Сохранить**, чтобы зафиксировать эту настройку.
:end

:ud_edit_script_choose_name_plug:
> Выберите имя своего сценария из раскрывающегося меню выбора файла или введите нужное название сценария.
:end

:ud_edit_user_script_choose_name_plug:
> Если у вас установлен плагин User Scripts, вы можете настроить не назначено Devices показывать состояние выполнения сценария пользователя, который будет использовать это устройство. Это поле будет включено только тогда, когда существует файл сценария.
:end

:ud_edit_script_run_background_plug:
> Включите "Запуск в фоновом режиме", чтобы сценарий запускался как фоновая задача. Это может предотвратить зависание интерфейса Unassigned Devices, когда сценарий запускается из пользовательского интерфейса.
:end

:ud_edit_script_buttons_plug:
> Нажмите кнопку **Сохранить**, чтобы внести текущие изменения. Нажмите **Сбросить**, чтобы отменить любые внесенные вами изменения (перед сохранением). Нажмите * * Готово**, чтобы выйти из этой страницы.
>
> Нажмите кнопку **Сценарий по умолчанию**, чтобы инициализировать поле редактирования сценарием по умолчанию.
> Вам все равно нужно нажать **Сохранить**, чтобы зафиксировать изменение.
:end
