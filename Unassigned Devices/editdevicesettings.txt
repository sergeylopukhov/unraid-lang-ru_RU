:ud_edit_script_device_settings_plug:
> Выберите параметры для этого устройства и точки монтирования.
:end

:ud_edit_script_detach_plug:
> Нажмите **Detach**, чтобы отсоединить устройство. После этого оно появится в Historical Devices и сможет быть подключено обратно из настроек устройства.
:end

:ud_edit_script_attach_plug:
> Нажмите **Attach**, чтобы подключить отсоединённое устройство. Оно будет обработано как только что установленное и автоматически смонтируется, если включён Auto Mount. Появление устройства в UD может занять некоторое время.
:end

:ud_edit_script_abort_plug:
> Нажмите **Abort**, чтобы прервать device script, user script и завершить все процессы, работающие в точке монтирования.
:end

:ud_disk_name_plug:
> Введите alias для устройства. Это имя будет использоваться для обращения к диску. Если оставить поле пустым, будет назначено имя 'devX'.
>
> **Name:** максимум 8 символов; пробелы заменяются на '_'; допускается минимум специальных символов; регистр учитывается.
>
> **Примечание:** имя не может начинаться с "dev", это зарезервировано для устройств 'devX'.
:end

:ud_edit_script_choose_name_plug:
> Это файл скрипта, который запускается при монтировании устройства. Выберите имя файла скрипта из выпадающего списка или введите нужный скрипт.
:end

:ud_edit_user_script_choose_name_plug:
> Если установлен плагин User Scripts, Unassigned Devices может показывать состояние выполнения User Script, который использует это устройство.
> Поле доступно только если существует Device Script File.
:end

:ud_edit_script_run_background_plug:
> Установите **Yes**, чтобы Device Script выполнялся в фоне. Это помогает избежать зависания UI Unassigned Devices при запуске скрипта из интерфейса.
:end

:ud_edit_script_buttons_plug:
> Нажмите **Default**, чтобы заполнить редактор стандартным Device Script. Чтобы применить изменение, всё равно нужно нажать **Save**.
>
> Нажмите **Save**, чтобы сохранить файл Device Script.
>
> Нажмите **Delete**, чтобы удалить файл Device Script.
>
> Нажмите **Reset**, чтобы отменить изменения до сохранения.
>
> Нажмите **Apply**, чтобы применить текущие правки.
>
> Нажмите **Done**, чтобы выйти со страницы.
:end
