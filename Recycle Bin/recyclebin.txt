:recycle_bin_shares_plug:
> Выберите общие ресурсы, для которых будет работать Recycle Bin. Настройки применяются к удалённым файлам внутри выбранных SMB-ресурсов.
:end

:recycle_bin_enable_plug:
> Установите **Yes**, чтобы при запуске сервера включать Recycle Bin для удалённых файлов.
>
> Чтобы скрыть папку Recycle Bin, включите параметр `Hide Dot Files` в настройках SMB.
:end

:recycle_bin_hide_folder_plug:
> Чтобы скрыть папку Recycle Bin, включите параметр `Hide Dot Files` в настройках SMB.
:end

:recycle_bin_background_plug:
> Запускает операции очистки в фоне, чтобы страница Recycle Bin не зависала на больших объёмах удалённых файлов.
:end

:recycle_bin_ud_plug:
> Включает работу Recycle Bin для устройств Unassigned Devices, если они опубликованы по SMB и поддерживают нужные параметры.
:end

:recycle_bin_parameters_plug:
> Дополнительные параметры Recycle Bin передаются в конфигурацию Samba. Меняйте их только если понимаете синтаксис vfs_recycle и последствия для удаления файлов.
:end

:recycle_bin_permissions_plug:
> Управляет правами создаваемых папок и файлов Recycle Bin. Настройка нужна, если пользователи не видят или не могут очищать свои удалённые файлы.
:end

:recycle_bin_excluded_files_plug:
> Список файлов, которые не будут попадать в Recycle Bin. Указывайте шаблоны по одному на строку или через поддерживаемый формат Samba.
:end

:recycle_bin_excluded_dirs_plug:
> Список каталогов, исключённых из Recycle Bin. Файлы, удалённые из этих каталогов, будут удаляться сразу.
:end

:recycle_bin_excluded_shares_plug:
> Список общих ресурсов, исключённых из Recycle Bin.
>
> Например, 'Squidbait' исключит все ресурсы, созданные плагином Ransom Ware.
>
> Примечание: исключение ресурса не удаляет уже существующие файлы в .Recycle.Bin; они всё ещё будут учитываться в размере и отображаться на вкладке 'Shares'.
:end

:recycle_bin_age_day_plug:
> Файлы старше заданного количества дней удаляются по расписанию или при нажатии кнопки **Remove Aged Files**.
:end

:recycle_bin_remove_age_files_plug:
> Расписание удаления устаревших файлов. Hourly выполняется каждый час ровно в начале часа, Daily - каждый день в 3:00, Weekly - каждое воскресенье в 3:00.
:end

:recycle_bin_age_files_notification_plug:
> Если выбрано **Yes**, при запуске задачи удаления старых файлов по расписанию будет отправлено уведомление.
:end

:recycle_bin_log_deleted_files_plug:
> Если выбрано **Yes**, удалённые файлы будут записываться в журнал.
:end

:recycle_bin_show_log_plug:
> Журнал Recycle Bin:
>
> Удалённые файлы отображаются в журнале как unlinked files.
> Нажмите **Clear**, чтобы очистить журнал.
:end
