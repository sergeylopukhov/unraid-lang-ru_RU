File Activity=деятельность файлов
File Activity Monitoring=отслеживание активности файлов
Start File Activity=начните действие с файлами
Stop File Activity Monitoring=остановить мониторинг активности файлов
Clear the File Activity log=Очистите журнал действия
Refresh log=обновить журнал
Enable File Activity Monitoring=Включить мониторинг активности файлов
Enable for Unassigned Devices=включить для неназначенных устройств
Enable for Cache and Pool Disks=Включить для дисков кэша и пула

:file_activity_loggging_plug:
> Журнал активности файла:
>
> Активность чтения/записи/изменения файлов отслеживается и регистрируется в массиве с помощью inotify и отражается на диске, дисках UD и кэше.
> Чтобы зарегистрировать активность на диске, нужно запустить активность файлов.
> File Activity предназначен для работы в течение короткого периода времени, чтобы вы могли проверить активность диска.
> Сервер с большой активностью файлов может заполнить пространство журнала.
> Каталог 'appdata' исключен.
>
> Примечание: ведение журнала активности файлов прекращается, если остановлен массив.
:end

:file_activity_enable_logging_plug:
> Установите "да", чтобы включить ведение журнала активности файлов при запуске сервера.
:end

:file_activity_enable_ud_plug:
> Установите значение "Да«, чтобы включить ведение журнала активности файлов для неназначенных устройств, если установлен плагин»неназначенные устройства".
:end

:file_activity_enable_cache_plug:
	> Установите «Да», чтобы включить ведение журнала активности файлов для дисков кэша и пула. Диски пула применяются только в Unraid 6.9.
:end

:file_activity_log_by_plug:
> Установите значение **Disk**, чтобы регистрировать активность на диске. Установите **Share**, чтобы регистрировать активность совместного доступа.
:end
