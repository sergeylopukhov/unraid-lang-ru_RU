Hide Deprecated Applications=Скрыть устаревшие приложения
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=Этот параметр скрывает устаревшие программы из списка доступных программ. Однако устаревшие программы все равно будут отображаться в разделах установленных и предыдущих программ.
Hide Incompatible Applications=Скрыть несовместимые приложения
This setting will hide applications that are listed as being incompatible with your version of Unraid=Этот параметр скроет программы, которые указаны как несовместимые с вашей версией Unraid
Allow install of second instance=Разрешить установку второго экземпляра
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Включает дополнительную кнопку на карточках и во всплывающих окнах. Она позволяет переустановить приложение со значениями шаблона по умолчанию. Используйте этот параметр только если нужно запустить два экземпляра одного приложения одновременно.
Automatically open the sidebar=Автоматически открывать боковую панель
Allows CA to automatically open the sidebar for more information when hovering over an application's icon=Разрешает CA автоматически открывать боковую панель с подробностями при наведении на значок приложения
Allow CA to check for updates to applications=Разрешить CA проверять обновления приложений
This setting enables CA to check for updates to applications Note that Community Applications will always check for updates to itself=Этот параметр разрешает CA проверять обновления приложений. Сам плагин Community Applications всегда проверяет собственные обновления.
Disable Mouse Wheel Page Changes=Отключить смену страниц колесом мыши
This setting disables using left or right on a mouse wheel to change the page=Отключает смену страницы наклоном колеса мыши влево или вправо
Mouse wheel threshold=Порог колеса мыши
If you tilt the mouse wheel left or right, the browser returned value will change Set the threshold to the value returned Note that different browsers and operating systems may return different values=Если наклонить колесо мыши влево или вправо, браузер вернёт значение. Задайте порог по возвращённому значению. Разные браузеры и ОС могут возвращать разные значения
This is a global value for all devices If it causes issues, then disable Mouse Wheel Page Changes=Это глобальное значение для всех устройств. Если возникают проблемы, отключите смену страниц колесом мыши
Allow CA to send any emergency notifications=Позволить CA отправлять экстренные оповещения
This setting allows CA to periodically scan your system and alert you to the presence of=Этот параметр позволяет CA периодически сканировать вашу систему и сообщать о наличие

Save CA debugging information=Сохраните информацию о отладке CA
Only enable this when advised to from the author of Community Applications=Включите это только по совету автора программ сообщества
